# Employee Management System (EMS)

A complete **Employee Management System** built using **Spring Boot**. This application helps organizations efficiently manage employees, departments, leaves, and attendance with a clean and user-friendly interface.

---

## ✨ Features

- Add, View, Update and Delete Employees
- Department Management
- Role-based Access Control (Admin, HR, Employee)
- Leave Request & Approval System
- Employee Search & Pagination
- Attendance Tracking
- Responsive Dashboard

---

## 🛠️ Tech Stack

- **Backend**: Java 17, Spring Boot 3, Spring Data JPA, Spring Security
- **Frontend**: HTML5, CSS3, JavaScript, Thymeleaf, Bootstrap 5
- **Database**: MySQL
- **Security**: JWT Authentication + Role-based Authorization
- **Build Tool**: Maven
---

### Prerequisites
- JDK 17 or higher
- MySQL Server
- Maven
